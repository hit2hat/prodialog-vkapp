VK App for ProDialog

## Technologies
*   React
*   Redux
*   Rematch
*   VK UI
*   VK Connect / VK Connect Promise

## How to run
*   Clone repo
*   `yarn` or `npm install`
*   `yarn start` or `npm start`
*   Your copy will be launch on `127.0.0.1:10888`
